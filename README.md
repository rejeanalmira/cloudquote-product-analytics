# CloudQuote Product Analytics Portfolio

Welcome to my Product Analytics portfolio.

This repository contains a collection of end-to-end analytics projects built around a fictional SaaS company called **CloudQuote**. The projects simulate the work of a Product Analyst supporting engineering and product teams through data-driven insights.

## Goals

- Strengthen SQL and MySQL skills
- Build interactive dashboards
- Practice product analytics techniques
- Demonstrate business storytelling with data
- Showcase work relevant to Product Analyst roles

## Tech Stack
